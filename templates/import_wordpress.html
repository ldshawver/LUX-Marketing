Import WordPress Users

Import users from your WordPress site with roles as tags and automatic membership-based segmentation.

How It Works:
  • WordPress roles (admin, editor, member, etc.) are imported as tags
  • Users with "member", "premium", "vip", or "administrator" roles → Member segment
  • All other users → Website Users segment
  • Source is marked as "wordpress" for easy filtering
Must have WordPress REST API enabled (usually public by default)
